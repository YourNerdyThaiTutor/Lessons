Lesson 14 Consonant clusters

Some consonants have friends

Sometimes you can combine 2 consonants together and say them at the same time

There are 2 types of consonant clusters: True cluster, Not True cluster.

True clusters are pronounced the way they are witten. The following table shows you which combinations are true clusters.

First consonat cluster example
ร ล ว
ก กร กล กว กราบ กลาง แกว่ง
ข ขร ขล ขว เขรอะ ขลาด ขวาน
ค คร คล คว คราม คลาน ควาย
ต ตร ตรง ตราด
ป ปร ปล ปราบ ปลวก
ผ ผล แผล
พ พร พล พราย พลาด

Anything else that is not in this table you would only pronounce the first letter.

For example,

จริง is pronounced like จิง

เศร้า is pronounced like เศ้า

สร้อย is pronounced like ส้อย

Special Not True cluster "ทร"

When ทร is written together although it is a not true cluster it is not pronounced with just ท, it changes the sound completely to a ซ or the "s" sound. For example, ทราย(sand) is pronounced like ซาย, ไทร(Ficus tree) is pronounced like ไซ, ทรง(shape) is pronounced like ซง There are 18 words in Thai that use ทร as ซ, which are the following.
